SEARCH STATE COVID INFORMATION

Welcome to the State Covid Tracker. This application enables you to search for up to date Covid-19 data in all 50 States. To search the current data, simply put two letters for the state in the first field, eg. 2 letter for the state "CA" if you want to get the current data for that state. You can also search by date in the second field, simply click on the field and calendar will pop up to choose the date.
